# CESI2527

## Description

Ce dépôt contient les ressources pédagogiques pour le cours essentiel d'infrastructure réseau.

### Contenu

- Documentation et guides
- Exemples de configuration
- Exercices pratiques
- Ressources de référence

### Objectifs du cours

Maîtriser les concepts fondamentaux de l'infrastructure réseau, incluant :
- Topologies réseau
- Protocoles de communication
- Sécurité réseau
- Administration réseau
